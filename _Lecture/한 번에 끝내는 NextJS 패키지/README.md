## 기술 스택

 - `Backend`
    - 언어 및 프레임워크: JavaScript, NestJS
    - 캐싱: Redis
    - 인증/인가: Passport
    - 통신: SocketIO, RabbitMQ, gRPC
    - 문서화: Swagger
    - 데이터베이스: TypeORM, Prisma, Postgresql, MongoDB
    - 큐: BullMQ
    - 로깅: Winston, Pino
    - 이벤트: Kafka
    - 테스트: Jest, Supertest
 - `DevOps`
    - 배포: AWS
    - 오케스트레이션: Kubernetes, Helm
    - 모니터링: Prometheus, Grafana
    - 인프라 스트럭처: Terraform
    - 서비스메쉬: Istio
    - 로깅: Elasticsearch, Fluentd, Kibana
