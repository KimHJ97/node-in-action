# 동시성 프로그래밍

## 1. 동시성 & 멀티쓰레딩

JavaScript는 기본적으로 싱글 스레드지만, 동시성은 지원한다.

### JavaScript는 싱글 스레드

JavaScript 엔진(예: V8)은 기본적으로 한 번에 하나의 작업만 실행한다.

```javascript
// 순서대로 실행된다. (동시 X)
console.log("A");
console.log("B");
console.log("C");
```

### 동시에 되는 것처럼 보이는 이유 (동시성)

JavaScript는 실제로는 하나씩 처리하지만, 비동기 처리 + 이벤트 루프 덕분에 동시에 처리하는 것처럼 보인다.

- Call Stack (실행 중인 함수)
- Web APIs (브라우저/Node 비동기 처리 영역)
- Callback Queue / Task Queue
- Event Loop

```javascript
// setTimeout은 Web API로 넘어감
// Call Stack이 비워진 후
// Event Loop가 Callback Queue에서 꺼내 실행

console.log("A");

setTimeout(() => {
  console.log("B");
}, 0);

console.log("C");
```

## 2. 싱글쓰레드

### 2-1. Main Thread

Main Thread는 JavaScript가 실제로 실행되는 유일한 스레드이다.

### 2-2. Web APIs (브라우저/Node 환경)

웹 API(Web Application Programming Interfaces)는 웹 브라우저가 자바스크립트를 통해 제공하는 내장 기능 집합으로, 웹 페이지가 운영 체제나 네트워크, 하드웨어 리소스와 안전하게 상호작용할 수 있게 한다. `setTimeout`, `fetch`, `localStorage` 등은 대표적인 예로, 현대 웹 애플리케이션 개발의 핵심 기반을 이룬다.

- setTimeout
- setInterval
- fetch / AJAX
- DOM 이벤트 (click 등)

```javascript
// setTimeout → Web API로 넘김
// 타이머 시작
// 완료되면 Queue로 이동
setTimeout(() => {
  console.log("B");
}, 1000);
```

### 2-3. Queue (대기열)

- **Callback Queue (Macrotask Queue)**
  - 일반적인 비동기 작업이 들어오는 곳
  - setTimeout
  - setInterval
  - DOM 이벤트
- **Microtask Queue**
  - 더 우선순위가 높은 큐
  - Promise.then
  - async/await

```javascript
Promise.resolve().then(() => console.log("microtask"));
```

## 3. 동기와 비동기

### 3-1. 동기 (Synchronous)

한 작업이 끝날 때까지 다음 작업을 실행하지 않음

- 순차 실행 (위 → 아래)
- 하나 끝나야 다음 실행
- 코드 흐름이 단순함
- 대신 느릴 수 있음

```javascript
console.log("1");
console.log("2");
console.log("3");

// 블로킹 발생 (3초 동안 아무것도 못함)
function sleep(ms) {
  const start = Date.now();
  while (Date.now() - start < ms) {}
}
sleep(3000);
console.log("끝");
```

### 3-2. 비동기 (Asynchronous)

작업을 기다리지 않고 다음 코드 먼저 실행

- 동시에 여러 작업 처리 가능
- 대기 시간 낭비 없음
- 대신 흐름이 복잡해질 수 있음

```javascript
console.log("1");

// setTimeout은 기다리지 않고 -> Web API로 넘겨버림
setTimeout(() => {
  console.log("2");
}, 1000);

console.log("3");
```

### 3-3. 비동기 처리 방법

- `Callback`

```javascript
setTimeout(() => {
  console.log("완료");
}, 1000);
```

- `Promise`

```javascript
fetch("/api")
  .then((res) => res.json())
  .then((data) => console.log(data));
```

- `async / await`

```javascript
async function getData() {
  const res = await fetch("/api");
  const data = await res.json();
  console.log(data);
}
```
