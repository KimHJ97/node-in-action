# 비동기 통신 제어와 응답일관성

## 1. debounce로 타이밍 제어하기

### 전략: 입력이 일시적으로 멈출때만 보내자

- 입력이 0.5초 동안 없을 때 보낸다.
- 0.5초 안에 새 입력이 들어오면, 이전 예약된 실행을 취소한다.

```javascript
let timeoutId;

function onInput(value) {
  clearTimeout(timeoutId);
  timeoutId = setTimeout(() => {
    console.log("실행:", value);
  }, 500);
}

onInput("a");
onInput("ab");
onInput("abc");

// 예시
export default function App() {
  const [title, setTitle] = useState();
  const timerRef = useRef(null);

  function inputChnageHandler(e) {
    const todoNum = +e.target.value;

    if (timerRef.current) {
      clearTimeout(timerRef.current);
    }

    const stID = setTimeout(() => {
      fetch(`https://jsonplaceholder.typicode.com/todo/${todoNum}`)
        .then((res) => res.json())
        .then((result) => setTitle(result.title));
    }, 500);
    timerRef.current = stID;
  }

  return (
    <div className="App">
      <h1>Hello CodeSandbox</h1>
      <input type="text" onChange={inputChangeHandler}></input>
      <div>title: {title} </div>
    </div>
  );
}
```

## 2. 데이터통신 요청 취소

### 2-1. AbortController

AbortController는 JavaScript에서 비동기 작업을 “취소(cancel)”할 수 있게 해주는 표준 API이다.

- `AbortController`: 취소를 관리하는 객체
- `AbortSignal`: 취소 신호를 전달하는 객체

```javascript
const controller = new AbortController();
const signal = controller.signal;

// fetch 취소 예시
const controller = new AbortController();

fetch("/api/data", {
  signal: controller.signal,
})
  .then((res) => res.json())
  .then((data) => console.log(data))
  .catch((err) => {
    if (err.name === "AbortError") {
      console.log("요청 취소됨");
    } else {
      console.error(err);
    }
  });

// 2초 뒤 취소
setTimeout(() => {
  controller.abort();
}, 2000);
```

### 2-2. 사용 사례

- `타임아웃 구현`

```javascript
function fetchWithTimeout(url, timeout = 3000) {
  const controller = new AbortController();

  const timer = setTimeout(() => {
    controller.abort();
  }, timeout);

  return fetch(url, { signal: controller.signal }).finally(() =>
    clearTimeout(timer),
  );
}
```

- `여러 요청 동시에 취소`

```javascript
const controller = new AbortController();

fetch("/api/1", { signal: controller.signal });
fetch("/api/2", { signal: controller.signal });

controller.abort(); // 둘 다 취소됨
```
